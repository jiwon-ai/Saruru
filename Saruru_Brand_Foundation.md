# Saruru — 브랜드 파운데이션

> 사르르(Saruru) = 한국어 의태어 "사르르"(부드럽게 녹아 사라짐). 의식은 *녹이기*. 색·목소리·로고 모두 이 한 단어에서 출발한다.

## 1. 정체성 (포지셔닝)

**한 줄:** 사르르는 사람을 상대하는 사람들을 위한 **마음의 응급처치** — 힘든 응대 직후, 감정이 쌓이기 전에 **사르르 녹여 식히는** 프라이버시-우선 도구.

- **누구에게:** 카페·식당·리테일·콜센터 등 고객 응대 종사자(1차: 한국 독립 카페·식당).
- **무엇을:** 사건을 적고 → 짧은 리프레임으로 다시 보고 → **사르르 녹여 보낸다**.
- **왜:** 작은 감정 잔여물이 쌓여 번아웃이 되기 전에, **즉시** 식힌다.
- **습관 레이어 (밤 의식):** 메인은 *즉각 해소*지만, **'자기 전 마음 비우기'**를 데일리 습관으로 권유한다. 매일 취침 전이라는 트리거가 산발적 사용(Risk A)을 보완. — "오늘 건 오늘 녹이고, 깨끗한 마음으로."

> ⚠️ **프레이밍 주의 (Risk H):** "응급처치"는 **은유**다. 의료·치료·진단을 *암시하지 말 것*. 카피는 "마음을 식히는 / 녹이는 / 내려놓는" 같은 **정서적** 표현으로. "치료한다 / 응급의료"식 단정 금지(App Store 의료 클레임 회피). 면책 문구는 항상 유지: *"의료·치료·상담·진단 서비스가 아닙니다."* 또한 **수면을 다룰 때 '불면 치료·수면 질 개선' 같은 의료 클레임 금지** — "마음을 비우고 잔다"는 정서적 표현만.

## 2. 색 — 미스트 블루

고요한 미스트 블루 — 차분·신뢰·맑음. (라벤더 후보를 거쳐 확정.) (이전 코랄/번트오렌지는 '불' 색이라 폐기.)

| 역할 | HEX |
|---|---|
| 배경 | `#F1F3F4` |
| 카드/표면 | `#FFFFFF` |
| **강조(메인)** | `#6E8AAE` |
| 연블루 틴트(칩·배지) | `#DDE6EE` |
| 보조 페일블루 | `#C7D6E5` |
| 본문 텍스트 | `#2B3038` |
| 보조 텍스트 | `#586070` |
| 다크 화면(녹이기) | `#2B3038` (불 검정 대신 딥 미스트 블루그레이) |

- **시그니처 그라데이션:** 미스트 블루 `#6E8AAE` → 페일블루 `#C7D6E5` 로 *사르르 풀리는* 그라데이션. 로고·녹이기 화면에 사용.
- 강조색은 절제해서 — 한 화면에 메인 강조 1개.

## 3. 보이스 & 톤

차분 · 보호적 · 정서적으로 정확 · 비복수적. **불/공격 이미지 배제.** 동사는 "녹이다 · 식히다 · 내려놓다 · 흘려보내다". "받아쳐라 / 복수" 같은 표현 금지.

## 4. 로고 / 비주얼 컨셉

- "사르르" 글자가 아래로 **사르르 녹아 번지며 사라지는** 형태. 또는 미스트 블루 방울이 풀리는 심볼.
- 핵심 애니메이션: 적은 글자가 **녹아 사라짐**(타서 재가 되는 연출 ✗).
- 아이콘: 불꽃(🔥) 대신 **방울/스파클/안개**.
- **타이포그래피:** 로고·제목 = **Gowun Dodum(고운돋움)**. 본문 UI도 동일 계열로 통일.

## 5. 슬로건

- **메인:** "쌓이기 전에, 녹여 보내요."
- **보조:** "마음을 식히는 3분."
- **프라이버시:** "잔여물은 녹이고, 프라이버시는 지킵니다."
- **밤 태그라인:** "오늘 건 오늘 녹이고, 깨끗하게 자요."
- EN: "Before it builds up, melt it away."
